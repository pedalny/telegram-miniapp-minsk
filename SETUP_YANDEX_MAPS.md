# Инструкция по получению API ключа Яндекс.Карт

## Шаг 1: Регистрация в Яндекс ID

1. Перейдите на сайт: https://yandex.ru/
2. Если у вас нет аккаунта Яндекс, нажмите **"Войти"** и создайте аккаунт
3. Если аккаунт есть, войдите в него

## Шаг 2: Переход в Кабинет Разработчика

1. Перейдите по ссылке: https://developer.tech.yandex.ru/
2. Нажмите **"Войти"** и авторизуйтесь через Яндекс ID

## Шаг 3: Создание приложения

1. В кабинете разработчика нажмите **"Создать приложение"** или **"Добавить приложение"**
2. Заполните форму:
   - **Название приложения**: `Подработки Минск` (или любое другое)
   - **Тип приложения**: Выберите **"Веб-сервис"** или **"Веб-приложение"**
   - **Описание**: `Telegram Mini App для поиска подработок в Минске`
3. Нажмите **"Создать"**

## Шаг 4: Получение API ключа

1. После создания приложения вы попадете на страницу управления
2. Найдите раздел **"JavaScript API и HTTP Геокодер"** или **"API ключи"**
3. Нажмите **"Создать новый ключ"** или **"Получить ключ"**
4. Выберите тип ключа: **"JavaScript API и HTTP Геокодер"**
5. Укажите домен (для локальной разработки можно указать `localhost` или `*`)
6. Нажмите **"Создать"**
7. Скопируйте полученный API ключ (выглядит как длинная строка символов)

## Шаг 5: Добавление ключа в проект

### Вариант 1: В файл .env (для бэкенда, если нужно)

Добавьте в файл `.env`:

```env
YANDEX_MAPS_API_KEY=ваш_ключ_здесь
```

### Вариант 2: В HTML файл (обязательно)

Откройте файл `frontend/index.html` и найдите строку:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_YANDEX_API_KEY&lang=ru_RU" type="text/javascript"></script>
```

Замените `YOUR_YANDEX_API_KEY` на ваш реальный ключ:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ваш_реальный_ключ&lang=ru_RU" type="text/javascript"></script>
```

## Шаг 6: Проверка работы

1. Запустите сервер
2. Откройте приложение в браузере
3. Должна загрузиться карта Минска с центром на город

## Лимиты и тарифы

- **Бесплатный тариф**: 25,000 запросов в день (достаточно для разработки и небольшого проекта)
- Для продакшена с большим трафиком может потребоваться платный тариф

## Проблемы и решения

### Карта не загружается, ошибка "Invalid API key"

**Решение**: 
- Проверьте, что ключ скопирован полностью и без лишних пробелов
- Убедитесь, что ключ добавлен в `frontend/index.html`
- Проверьте, что домен в настройках ключа соответствует вашему домену (или используйте `*` для разработки)

### Карта серая, без деталей

**Решение**: 
- Проверьте подключение к интернету
- Убедитесь, что API ключ активен в кабинете разработчика
- Проверьте консоль браузера на наличие ошибок (F12)

### Ошибка "API key is not valid for this domain"

**Решение**: 
- В настройках ключа в кабинете разработчика добавьте ваш домен
- Для локальной разработки добавьте `localhost` или `*`
