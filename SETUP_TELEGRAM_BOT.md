# Инструкция по созданию Telegram бота

## Шаг 1: Создание бота через BotFather

1. Откройте Telegram и найдите бота **@BotFather**
2. Нажмите **"Start"** или отправьте команду `/start`
3. Отправьте команду `/newbot`
4. BotFather попросит ввести **имя бота** (отображается в списке контактов):
   ```
   Подработки Минск
   ```
5. Затем попросит ввести **username бота** (должен заканчиваться на `bot`):
   ```
   minsk_jobs_bot
   ```
   Или любой другой свободный username, например: `minsk_podrabotki_bot`

6. Если username свободен, BotFather выдаст вам **токен бота**, например:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```

## Шаг 2: Сохранение токена

**ВАЖНО**: Сохраните токен в безопасном месте! Он понадобится для настройки приложения.

Скопируйте токен и добавьте его в файл `.env`:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

## Шаг 3: Настройка бота (опционально)

Вы можете настроить описание и аватар бота:

1. Отправьте `/setdescription` и выберите вашего бота
2. Введите описание, например: "Поиск подработок в Минске"
3. Отправьте `/setabouttext` и выберите вашего бота
4. Введите краткую информацию о боте

## Шаг 4: Создание Mini App (после запуска сервера)

После того, как вы запустите сервер и он будет доступен из интернета:

1. Отправьте BotFather команду `/newapp`
2. Выберите вашего бота из списка
3. Введите название приложения: `Подработки Минск`
4. Введите описание: `Поиск подработок в Минске`
5. Введите URL вашего приложения (после запуска сервера):
   - Для локальной разработки через ngrok: `https://ваш_ngrok_домен.ngrok.io`
   - Для продакшена: `https://ваш_домен.com`
6. Загрузите иконку (опционально, размер 640x360px)

## Шаг 5: Тестирование бота

1. Найдите вашего бота в Telegram по username (например, `@minsk_jobs_bot`)
2. Нажмите **"Start"**
3. Если Mini App настроен, вы увидите кнопку для открытия приложения
4. Нажмите на кнопку - должно открыться ваше приложение

## Проблемы и решения

### Бот не отвечает

**Решение**: Проверьте правильность токена в `.env` файле и перезапустите сервер

### Mini App не открывается

**Решение**: 
- Убедитесь, что сервер доступен из интернета (используйте ngrok для локальной разработки)
- Проверьте, что URL начинается с `https://`
- Проверьте, что сервер отвечает на запросы (откройте URL в браузере)

### Ошибка "Invalid token"

**Решение**: Проверьте, что токен скопирован полностью и без лишних пробелов

## Для локальной разработки: Использование ngrok

Если вы разрабатываете локально, используйте ngrok для доступа из интернета:

1. Скачайте ngrok: https://ngrok.com/download
2. Запустите сервер на порту 8000
3. В другом терминале запустите:
   ```bash
   ngrok http 8000
   ```
4. Скопируйте HTTPS URL (например, `https://abc123.ngrok.io`)
5. Используйте этот URL при настройке Mini App в BotFather
